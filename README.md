# AI Resume Analyzer

## Overview
AI Resume Analyzer is an AI-powered application that analyzes resumes and provides personalized feedback to help users improve their chances of getting shortlisted for jobs. The project uses Google's Gemini API and Natural Language Processing (NLP) techniques to evaluate resume content and generate intelligent suggestions.

## Features
- Upload and analyze resumes
- AI-powered resume evaluation
- Skill extraction and keyword analysis
- ATS-friendly resume recommendations
- Personalized improvement suggestions
- User-friendly interface

## Technologies Used
- Python
- Google Gemini API
- Natural Language Processing (NLP)
- Streamlit
- PDF Processing Libraries

## How It Works
1. User uploads a resume in PDF format.
2. The application extracts text from the resume.
3. Resume content is sent to Gemini API for analysis.
4. AI evaluates skills, keywords, strengths, and weaknesses.
5. Personalized suggestions and feedback are generated.

## Use Cases
- Resume Improvement
- ATS Optimization
- Skill Gap Identification
- Career Guidance
